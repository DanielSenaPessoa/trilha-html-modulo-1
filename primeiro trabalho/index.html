HTML

(HyperText Markup Language)
é a linguagem padrão usada para criar páginas na web. Ele é responsável pela estrutura e pelo conteúdo das páginas, permitindo que os navegadores interpretem e exibam textos, imagens, links, vídeos e outros elementos de mídia.
